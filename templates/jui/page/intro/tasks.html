Task

Tasks are created by Project Manager. Normally tasks would be the same list like client would receive on quote or estimation. Tasks can take from few days up to few weeks to be completed.

Development Schedule

Project manager, who have created the tasks can also prioritize them. This makes it easy for developers to see what they need to work on at any given point in time. For clients it's important to understand that software is being build gradually and at early stages it might be impossible to try software out.
