Time Entry

All the members of the team enter time they have worked into the system on a regular basis. It is very improtant that you enter time as soon as possible. Colubris has many different ways of entering timesheets which fall into 3 categories:

  • 1. Automated Time Entry — Time Tracking application uses URL to publish
  • 2. Manual entry of completed tasks
  • 3. Import data through several supported formats

Automated Time Entry

Data should be sent either through GET or POST request to the following URL:

Arguments and values:

  • title (mandatory) — Title of the completed task
  • date (mandatory) — Date of the task
  • minutes (mandatory) — How many minutes have you worked
  • start — Time when started work in format hh:mm
  • end — Time when started work in format hh:mm
  • budget_id - Automatically set budget, if you know ID
